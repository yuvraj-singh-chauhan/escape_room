# Python Escape Room Game

A web-based escape room game themed around Python programming puzzles. Solve tricky Python questions within a 5-minute time limit to escape!

## How to Play

1. Open `index.html` in a web browser or host it on a web server (like GitHub Pages).
2. Click 'Start Adventure' to begin.
3. Answer 10 Python-related questions, with 3 attempts per question.
4. Score at least 5 to escape, aim for 7+ for mastery.
5. You have 5 minutes totalâ€”watch the timer!

## Hosting on GitHub Pages

1. Push this repository to GitHub.
2. Go to the repository settings, under 'Pages', set the source to the `main` branch.
3. Access the game via the provided GitHub Pages URL (e.g., `username.github.io/repo-name`).

## Files

- `index.html` - Main game page
- `styles.css` - Styling for the game
- `script.js` - Game logic

## Features

- Modern, dark-themed UI with progress indicators
- 5-minute countdown timer
- Hints available for puzzles
- Responsive design for various screen sizes

Enjoy the challenge!
